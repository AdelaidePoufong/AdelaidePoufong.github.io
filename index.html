ITM 2nd Year Project
Cinema Management

A team project involving the development of a cinema management system where clients can subscribe to the system to receive information about movies. Starting time, end time, title, cinema location, screen number and much more. The information is published from a cloud container to one or more users depending on the information requested. Any information in the database is push up using a mqtt script and this information is in return display on a website where user can filter or search for anything they are interested in.

Technologies & Tools:
  • Superbase for database
  • Google cloud for the mqtt script
  • Java script and html 5 for the webpage
  • Bash scripts
  • Dev.azure for collaboration
  • HTML5
